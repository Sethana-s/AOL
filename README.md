# Advice of Life (AOL)

Welcome to the Advice of Life (AOL) project repository. This project provides a comprehensive website for personal risk insurance advice, helping users understand their options and take the first step towards the right coverage.

## Features
- **Home Page**: Overview of the services offered.
- **About Us**: Information about the organization.
- **Services**: Detailed descriptions of the services provided.
- **FAQs**: Frequently Asked Questions to assist users.
- **Testimonials**: Feedback from satisfied clients.
- **Contact Page**: Get in touch with the team.

## Technologies Used
- **HTML5**: For structuring the content.
- **CSS3**: For styling the website.
- **JavaScript**: For interactivity.
- **Bootstrap**: For responsive design.

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/O365DevHero/AOL.git
   ```
2. Navigate to the project directory:
   ```bash
   cd AOL
   ```
3. Open `index.html` in your browser to view the website.
